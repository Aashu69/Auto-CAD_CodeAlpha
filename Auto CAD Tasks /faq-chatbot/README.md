A simple Python-based FAQ Chatbot that matches user questions to the most relevant FAQ using NLP and cosine similarity.

## Features
- Preprocessing with NLTK
- TF-IDF vectorization
- Cosine similarity matching
- Tkinter chat interface

## Installation
```bash
git clone https://github.com/yourusername/faq-chatbot.git
cd faq-chatbot
pip install -r requirements.txt
```

## Running
```bash
python chatbot.py
```
